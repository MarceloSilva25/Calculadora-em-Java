# Calculadora-em-Java
Projeto de calculadora em Java desenvolvida em console, com foco no aprendizado de estruturas fundamentais da linguagem. O sistema realiza operações matemáticas básicas, utiliza switch-case para seleção de operações, do while para repetição controlada do programa e valida entradas inválidas, como divisão por zero.

English version <br>
This Java calculator project, developed for console execution, focuses on teaching fundamental language structures. The system performs basic mathematical operations, uses switch-case for operation selection, a do-while loop for controlled program repetition, and validates invalid inputs, such as division by zero.
